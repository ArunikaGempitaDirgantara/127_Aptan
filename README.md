# 127_Aptan
Penjelasan Project :
Aptan merupakan aplikasi untuk membeli tanaman dan berisi pengetahuan tentang tanaman. Menggunakan Firebase.
1. Sign Up : Sebelum masuk ke aplikasi, jika user belum mempunyai akun di aplikasi, maka dapat membuat akun terlebih dahulu.
2. Login : Jika user sudah punya akun, maka hanya tinggal login.
3. Halaman Informasi Tanaman : User dapat melihat informasi tentang tanaman.
4. Halaman Pembelian : User dapat add pesanan dan batal pesanan
5. Halaman Profil : Logout
